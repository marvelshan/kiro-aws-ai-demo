# UI 設計更新說明

## 設計理念

全新的 UI 設計參考了業界頂尖網站的設計風格，包括：
- **Medium** - 優雅的排版和閱讀體驗
- **Notion** - 簡潔現代的介面設計
- **Stripe** - 精緻的視覺細節和動畫效果

## 主要改進

### 1. 色彩系統
- 採用高級的黑白灰色調為主
- 藍色作為強調色（#0066ff）
- 漸層背景增加層次感
- 精心調整的陰影系統（4 個層級）

### 2. 排版優化
- 更大的字體尺寸，提升可讀性
- 優化的行高和字間距
- 標題使用負字間距（letter-spacing）增加現代感
- 使用系統字體堆疊，確保各平台最佳顯示

### 3. 視覺效果
- 毛玻璃效果的固定導航列（backdrop-filter）
- 流暢的過渡動畫（cubic-bezier）
- 懸停時的微互動效果
- 漸層背景和陰影增加深度

### 4. 卡片設計
- 圓角更大（12-16px）
- 多層次陰影系統
- 懸停時的提升效果
- 邊框增加精緻感

### 5. 程式碼區塊
- 深色主題的程式碼區塊
- 漸層背景增加質感
- 更好的語法高亮（Atom One Dark）
- 行內程式碼使用粉紅色強調

### 6. 按鈕和互動元素
- 更大的點擊區域
- 懸停時的提升動畫
- 柔和的陰影效果
- 清晰的視覺反饋

### 7. 響應式設計
- 完整的移動端優化
- 流暢的斷點過渡
- 觸控友好的元素尺寸

## 技術細節

### CSS 變數系統
```css
--primary-color: #1a1a1a
--accent-color: #0066ff
--shadow-sm/md/lg/xl: 4 個層級的陰影
--radius-sm/md/lg: 3 個層級的圓角
--transition: 統一的過渡效果
```

### 動畫效果
- 使用 `cubic-bezier(0.4, 0, 0.2, 1)` 實現流暢動畫
- 懸停時的 `translateY` 和 `scale` 效果
- Modal 的滑入動畫

### 視覺層次
1. 背景漸層
2. 卡片陰影
3. 懸停效果
4. 互動反饋

## 預覽方式

```bash
# 建置專案
npm run build

# 啟動本地伺服器
npx http-server dist

# 在瀏覽器開啟
open http://localhost:8080
```
